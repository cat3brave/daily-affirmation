# App Checklist / 動作確認チェックリスト

このファイルは、Daily Affirmation / 心のお守りアプリの動作確認項目をまとめたチェックリストです。

大きな修正をしたあと、デプロイ前、Supabase設定を変更したあとに確認します。

---

## 1. 基本起動

### 確認コマンド

```bash
npm run lint
npm run dev
```

### チェック項目

- [ ] `npm run lint` が 0 errors / 0 warnings で通る
- [ ] `npm run dev` で開発サーバーが起動する
- [ ] `http://localhost:3000` が開ける
- [ ] 赤いエラー画面が出ない

---
